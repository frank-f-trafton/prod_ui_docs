base/button

A plain skinned button.

User Interaction

  • Mouse: Press and release with the cursor overlapping the widget to activate.

  • Keyboard: Press return or kpenter to activate.

Callbacks

wid_buttonAction

Called when the button is activated.

Arguments

  • self: The widget.
